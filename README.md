# BiteMatch

A small, static browser app that turns vague cravings into meal suggestions using TheMealDB API.

## Run locally

1. Open `index.html` in your browser.
2. Alternatively, serve the folder as a static site using a simple local server.

## Features

- Free-text craving input
- Sample craving chips
- Keyword-based craving interpretation
- Meal recommendations from TheMealDB
- Loading state and graceful error handling
- Meal details modal
- Favorites and recent searches persisted in `localStorage`

## Files

- `index.html` — page structure and UI layout
- `styles.css` — visual styling and responsive design
- `script.js` — craving parsing, API logic, and interaction handling
